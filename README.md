
# Project overview

This project implements a solution to the Reader–Writer synchronization problem using semaphores.
The goal is to allow multiple readers to read shared data simultaneously while ensuring writers have exclusive access.

## Languages used

[![My Skills](https://skillicons.dev/icons?i=java&perline=10)](https://skillicons.dev)


## Technologies & Tools

- Java Threads
- Custom Semaphore implementation
- Console-based output

## Main files

- `Semaphore.java` – Semaphore implementation
- `ReadWriteLock.java` – Synchronization logic
- `Test.java` – Creates reader and writer threads
